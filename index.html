DriveBox — Privacy Policy

Last updated: 2026-07-05 · App: DriveBox (com.eltscode.drivebox) · Developer: Eltscode

DriveBox is built for offline entertainment on a car head unit. We respect your privacy.

1. Anonymous statistics

DriveBox sends anonymous install statistics so we can count how many devices use the app and on which versions. This consists only of: a random per-install ID (not a device or advertising ID, not tied to your identity, and reset if you reinstall), your device type (brand, model, Android version), the app version, and language. It contains no personal data, files, contacts, or location. It is used only to count installs — never for advertising or profiling. You can turn it off anytime in Settings → About → Send anonymous statistics.

2. Personal data

Apart from the anonymous statistics above, DriveBox does NOT collect, transmit, or share your personal data. No accounts, no ads.

2a. Network use

DriveBox is offline-first — all media playback and management work without internet. Besides the anonymous statistics above, the app connects to the internet only to check for and download app updates and to verify purchases, both via Google Play. No personal data is sent in the process.

3. Media files

The app reads video/audio from USB or head unit storage only to play and copy them at your command. Files stay on your device.

4. Location (GPS)

Location permission is optional and used only to show speed and trip data in the "Car Data" menu. Location is processed on-device and is NEVER sent anywhere.

5. Purchases

The one-time purchase is handled by Google Play Billing. DriveBox does not store your card/payment data.

6. Permissions

Storage: open media files. Location: speedometer. All optional and revocable via Android Settings.

7. Contact

Email: eltscode.dev@gmail.com

DriveBox — Kebijakan Privasi

Diperbarui: 2026-07-05

DriveBox dibuat untuk hiburan offline di head unit mobil. Kami menghormati privasi Anda.

1. Statistik anonim

DriveBox mengirim statistik pemasangan yang anonim agar kami tahu berapa banyak perangkat yang memakai aplikasi dan versi apa saja. Isinya hanya: sebuah ID acak per-instalasi (bukan ID perangkat atau iklan, tidak terkait identitas Anda, dan berubah bila Anda memasang ulang), jenis perangkat (merek, model, versi Android), versi aplikasi, dan bahasa. TIDAK ada data pribadi, file, kontak, atau lokasi. Data ini hanya dipakai untuk menghitung jumlah pemasangan — bukan untuk iklan atau pemrofilan. Anda bisa mematikannya kapan saja di Setelan → Tentang → Kirim statistik anonim.

2. Data pribadi

Selain statistik anonim di atas, DriveBox TIDAK mengumpulkan, mengirim, atau membagikan data pribadi Anda. Tidak ada akun maupun iklan.

2a. Penggunaan jaringan

DriveBox bersifat offline-first — semua pemutaran dan pengelolaan media berjalan tanpa internet. Selain statistik anonim di atas, aplikasi terhubung ke internet hanya untuk memeriksa & mengunduh pembaruan aplikasi dan memverifikasi pembelian, keduanya lewat Google Play. Tidak ada data pribadi yang dikirim.

3. File media

Aplikasi membaca video/audio dari USB atau penyimpanan head unit hanya untuk diputar dan disalin atas perintah Anda. File tetap berada di perangkat Anda.

4. Lokasi (GPS)

Izin lokasi bersifat opsional dan hanya dipakai untuk menampilkan kecepatan dan data perjalanan di menu "Data Mobil". Data lokasi diproses di perangkat dan TIDAK pernah dikirim ke mana pun.

5. Pembelian

Pembelian sekali bayar diproses oleh Google Play Billing. DriveBox tidak menyimpan data kartu/pembayaran Anda.

6. Izin

Penyimpanan: membuka file media. Lokasi: speedometer. Semua opsional dan bisa dicabut lewat Setelan Android.

7. Kontak

Email: eltscode.dev@gmail.com
